Mission IM possible

  • Controls
    • Mouse Click - move character
    • Mouse Drag - move map
    • key '1' - control marine character
    • key '2' - control spy character
    • key '3' - control sniper character
    • key 'z' + mouse click - shoot primary weapon
    • key 'x' + mouse click - shoot secondary weapon
    • Shift key - special action (Marine: Crawling, Sniper: Hiding, Spy: camouflage
    • key 'w', 'a', 's', 'd' - move map

  • Mission types
    • Clear the map
      • Kill all enemies
    • Obtain confidential document
      • Get confidential document in enemy's camp and return to the starting position
      • Only spy character can obtain and bring the confidential document
    • Explode target
      • Explode target on the map by throwing hand grenade
      • Only marine character can throw grenade
    • Assassination
      • Assassinate enemies' general
      • Only sniper can shot general because of shooting range
